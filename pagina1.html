Acessibilidade na Web


Por que a acessibilidade é importante?

Cerca de 23,9% dos brasileiros possuem algum tipo de deficiência. Para essas pessoas, a web pode ser uma ferramenta de autonomia – desde que seja inclusiva.

Barreiras comuns:

  • Imagens sem descrição.
  • Falta de legendas em vídeos.
  • Links genéricos como "Clique aqui".

Dicas:

  • Sempre use descrições para imagens (atributo alt).
  • Evite usar apenas cor para transmitir informações.
