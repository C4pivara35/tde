Exemplos Práticos: Como Fazer Certo


Títulos Estruturados

Use uma hierarquia lógica para os títulos, garantindo clareza e navegabilidade:

  • Certo: Use <h1> para o título principal e <h2>, <h3> para subtítulos.
  • Errado: Pular níveis ou usar títulos de forma desordenada.

Contraste de Cores

Certifique-se de que o texto seja legível, mesmo para pessoas com baixa visão:

  • Certo: Texto preto em fundo branco ou amarelo em preto.
  • Errado: Texto cinza claro em fundo branco.

Imagens

Todas as imagens relevantes devem incluir descrições alternativas:

  • Certo: <img src="foto.jpg" alt="Descrição da foto">
  • Errado: <img src="foto.jpg"> (sem o atributo alt).

Formulários

Formulários precisam ser claros e acessíveis:

  • Certo: Inclua <label> para cada campo do formulário e use descrições claras.
  • Errado: Indicar campos obrigatórios apenas com cores.

Design Interativo

Crie exemplos no site para ensinar boas práticas. Por exemplo:

  • Simule diferentes níveis de contraste para que os usuários entendam a diferença.
  • Mostre a navegação com leitores de tela para reforçar a importância de descrições e hierarquias corretas.
